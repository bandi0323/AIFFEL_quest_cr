# AIFFEL Quest Repository

AIFFEL 과정에서 수행한 퀘스트와 프로젝트를 정리하는 레포지토리입니다.  
각 폴더는 과정별 학습 내용과 퀘스트 결과물을 구분하여 관리합니다.

## Folder Description

- MainQuest: 메인 퀘스트 과제 정리
- Python: Python 관련 퀘스트 정리
- Flutter: Flutter 관련 퀘스트 정리
- Exploration: Exploration 프로젝트 정리

## Rule

각 퀘스트 폴더에는 다음 파일을 포함합니다.

- .ipynb: 프로젝트 또는 실습 노트북 파일
- README.md: 피어리뷰 및 회고 정리 파일
